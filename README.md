🛒 Grocery Sales Analysis Dashboard
📌 Overview

This project is a data visualization dashboard created using Power BI Desktop. It focuses on analyzing overall business performance over a period of time using interactive visuals and key metrics.

🎯 Objectives
To analyze sales, profit, and order performance
To track growth trends over multiple years
To compare current performance with previous periods
To support decision-making through data insights

🛠️ Tools Used
Power BI Desktop
Data Cleaning & Transformation
DAX (Data Analysis Expressions)

📊 Key Features
Interactive dashboard with filters
Time-based analysis (YTD, MTD, QTD)
Performance comparison with targets
Region-wise and customer-level insights

📈 Key Metrics
Total Sales
Net Sales
Total Profit
Total Orders

📂 Dataset
The dataset includes information related to sales, orders, time period, and regional performance, which helps in analyzing trends and patterns.
